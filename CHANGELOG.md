# Change Log

All notable changes to RemoteKit are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-06-20

### Changed

- Packaging and Marketplace listing metadata refinements. No functional changes.

## [0.1.0] - 2026-06-20

Initial release.

### Added

- **Remote File Explorer** — browse the remote filesystem independently of the
  opened workspace folder, with a clickable breadcrumb, lazy per-folder loading,
  a virtualized tree, file open-in-preview, and re-rootable folders
  (**Set as Explorer Root** / **Set as Files Root**, plus parent / reset /
  refresh title-bar actions).
- **Gallery view** — small / medium / large thumbnail densities, with image
  thumbnails and file-type glyph fallbacks.
- **Uploads** — paste, an upload button, and ⇧ Shift + drag, with
  Overwrite / Keep Both / Skip conflict handling.
